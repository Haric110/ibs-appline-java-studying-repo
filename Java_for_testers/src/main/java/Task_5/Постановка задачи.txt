Написать калькулятор, используя принципы ООП.

Калькулятор - устройство для вычислений математических выражений.
В программу Калькулятор вводятся аргументы и выбираются операторы между ними.
Операторы могут быть:
    - унарными (инверсия, обращение, извлечение корня, возведение в квадрат или куб,  тригонометрические функции)
    - бинарными (сложение, вычитание, умножение, деление, возведение в определённую степень)
Оператор возвращает значение результата операции (функции).
Результат операции может быть передан в качестве левого аргумента для бинарного оператора или в качестве единственного
 аргумента унарного оператора.
Возможна реализация скобок и вложенных вычислений.
Скобки можно реализовать как оператор, который последовательно вызывает другие операторы и возвращает результат цепочки.

Как реализовать цепочку вычислений...

Объект калькулятора хранит промежуточное значение результата.
Функции операторов могут вызываются как из объекта калькулятора, так и статически.

Простая цепочка.
Сначала инициализируется первичный аргумент он же результат вычислений. Значение по умолчанию 0, или инициализируется
  в конструкторе с аргументом или сеттером.
Затем выбирается оператор.
Если оператор бинарный, то его левый аргумент принимает значение первичного аргумента, а правый инициализируется
  аргументом реализующей его функции.
Если оператор унарный, то его единственный аргумент принимает значение первичного аргумента, реализующая функция без
  аргументов.
Результат операции сохраняется в результат вычислений вместо первичного аргумента. Следующие операторы используют его
аналогично описанным выше способом.

При выборе скобок должен создаваться новый объект цепочки вычислений. То есть в рамках данной задачи скобки
  реализовывать не требуется.
